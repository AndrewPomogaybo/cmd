Используя интегрированную среду разработки (IDE) Microsoft Visual Studio требуется разработать консольную программу-утилину в соответствии с заданием на соответствующую итоговую оценку
Для разработки допускается использовать только язык программирования C#. Использовать любой другой язык программирования запрещено.
Задания необходитмо выполнять строго последовательно, начиная с оценки в 3 балла. Перескакивать сразу на оценку в 5 баллов нельзя.
Реализацию каждой команды необходимо выполнять в отдельном файле проекта. Обязательно использовать ООП подход. Для каждой команды обязательно выполнить "обертку" кода в класс с несколькими методами.
В коде проекта обязательно должны быть реализованы документирующие комментарии.
Для каждой утилиты необходимо реализовать свой набор входных аргументов в количестве 3 штук. Аргументы --help / -h необходимо реализовать в каждой команде отдельно, он является обязательным.
Использовать любые сторонние подключаемые библиотеки запрещено.
